# indirekte_rede_aktiv_passiv

`indirekte_rede_aktiv_passiv` ist ein statisches GitHub-Pages-Repo für den Deutschunterricht. Die Lernumgebung bündelt eine umfassende Übungswerkstatt zu indirekter Rede sowie Aktiv und Passiv in drei klar gestuften Levels.

## Was enthalten ist

- `Basis` mit sicheren Grundmustern, einfachen Pronomenwechseln und klaren Aktiv-Passiv-Umbauten
- `Erhöhte Ansprüche` mit Fragen, Aufforderungen, Modalverben, Vorzeitigkeit und Passivvarianten
- `Experte` mit komplexen Sprecherperspektiven, kniffligen Deixiswechseln und mehrschrittigen Textrevisionen
- drei Themenbereiche pro Level:
  - indirekte Rede
  - Aktiv/Passiv
  - kombinierte Aufgabenstellungen
- zusätzliche Multiple-Choice-Aufgaben mit gemischter Antwortreihenfolge
- insgesamt 63 kuratierte Aufgaben
- Filter nach Level, Thema und Format
- Zufallsaufgabe für spontane Trainingsphasen
- lokaler Fortschritt direkt im Browser
- Sofortkorrektur mit drei Fehlversuchen:
  - erster Fehler: nur Rückmeldung `falsch`
  - zweiter Fehler: Rückmeldung plus gezielter Hinweis
  - dritter Fehler: Musterlösung, Erklärtext, farbige Pfeile und Umbau-Schemata
- versteckbare beziehungsweise automatische Musterlösungen für Selbstkontrolle oder Unterrichtsgespräch

## Projektstruktur

- `index.html` - Oberfläche und Seitenstruktur
- `styles.css` - Gestaltung, Layout und Responsive Design
- `data.js` - kompletter Aufgabenpool mit Theorie- und Lösungsteilen
- `app.js` - Filterlogik, Fortschrittsspeicherung und Rendering
- `GITHUB_METADATA.md` - Vorschläge für Repository-Beschreibung und Topics
- `.github/workflows/pages.yml` - einfacher Workflow für GitHub Pages

## Didaktische Schwerpunkte

- sichere Bildung von Konjunktiv I und sinnvolle Ausweichformen mit Konjunktiv II oder `würde`
- systematische Pronomenwechsel bei Sprecher- und Perspektivverschiebung
- Umgang mit Zeit- und Ortsangaben wie `morgen`, `gestern`, `hier`, `dort`
- Aktiv-Passiv-Umbau in Präsens, Präteritum und zusammengesetzten Zeiten
- Vorgangspassiv, Zustandspassiv, unpersönliches Passiv und Modalverbpassiv
- kombinierte Umbauten mit indirekter Rede und passivischer Neuformulierung
- tolerant-strenge Antwortprüfung mit akzeptierten Varianten
- Multiple Choice, Umformen, Lückentext, Fehlerdiagnose, Frage/Aufforderung, Textrevision und Mehrschritt

## Lokal nutzen

1. Den Ordner als eigenständiges Repository auf GitHub anlegen oder in ein bestehendes Repo kopieren.
2. `index.html` direkt im Browser öffnen.
3. Oder GitHub Pages aktivieren; der enthaltene Workflow ist dafür vorbereitet.

## Einsatzideen

- als gestufte Werkstatt im Wochenplan
- als Training vor einer Grammatikprüfung
- als Materialpool für binnendifferenzierten Unterricht
- als Selbstlernangebot mit Musterlösungen
